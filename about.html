About Soteria Network (SOTER)

Your trusted cryptocurrency for empowering holding and investments, safeguarding with cutting-edge technology. Building the future of Web3 infrastructure with GPU friendly PoW blockchain technology.

Coin Specifications

General Information

  • Network Type: Layer 1. Independent blockchain network
  • Coin Name: Soteria
  • Ticker Symbol: SOTER
  • Algorithms: SoterG
  • Block Size: 3 MiB max
  • TX Size: 2 MiB max
  • Maximum Supply: 6,000,000 Soter after 30 years
  • Codebase: Completely adapted to handle 10s block time
  • Blockchain: UTXO model
  • TX Fee: 0.001 Soter/kb
  • TX Confirmation: 6
  • Smart Contract: Still in testnet using Lua VM
  • Sync Time: Improved upto 3 times than other traditional blockchains
  • Ports: P2P Port: 8323, RPC Port: 7896
  • Node: Min 4GiB RAM, Recommended 8GiB RAM

Block Reward

  • Block Reward: 0.14 Soter/Block
  • Block Time: 12 Seconds average
  • Orphan Rates: Decreased upto 70% than other blockchains that have the same block time
  • Miners Reward: 0.042 Soter/Block
  • Nodes Reward: 0.0028 Soter/Block
  • Miners Emission: ~300 daily, ~9000 monthly, ~110K yearly, this is valid only for 1st year
  • Nodes Emission: ~20 daily, ~600 monthly, ~7250 yearly, this is valid only for 1st year
  • Dev Emission: ~50 daily, ~1500 monthly, ~18250 yearly, this is valid only for 1st year
  • Difficulity: LWMA3-EMA, retarget at each 90 blocks
  • Dev Fee: 0.007 Soter/Block
  • Coinbase Maturity: 4200 blocks / ~14 hours
  • Halving: Mining reward will be reduced by 3.5% each 3000000 blocks or by 3.33% each year
  • Total Emission: ~1000 daily, ~30K monthly, ~365K yearly, this is valid only for 1st year
  • Burning Mechanism: Algorithmic burning mechanism will be activated at 2M height
